Enquanto o poste dá a luz por cima, nós desenvolvedores estamos nos esforçando para dar a luz por onde der, pois esse blog está realmente um parto complicado!

Juntando uma velha vontade com a necessidade de se implementar algo para uma disciplina de Engenharia de Software, estamos trazendo ao mundo dos vivos a primeira versão alpha do Poste.

O objetivo é criar uma engine minimalista capaz de sustentar um blog médio com o que há de mais moderno em termos de tecnologia de aplicações para a web.

O quesito minimalismo é o aspecto mais atendido até o momento, pois ainda não foi implementada nem a interface de edição de posts nem a possibilidade de adição de comentários. é, por enquanto só vai mostrar os posts que já estiverem no banco de dados...

Não se assutem, mesmo para cinco destemidos desenvolvedores é complicado manter em dia a vida real, a acadêmica e ainda tocar esse projeto.

O Que vem por aí?

A resposta curta é: Muita Coisa! Não estamos muito afim de re-escrever coisas que já existem. Já gastamos nossa cota de re-invenção da roda (quase) criando uma engine para blog, portanto pretendemos integrar o blog con serviços como o Disqus para comentários, github.com para code snippets, melhorar nosso sitema de temas para que err... funcione, entre outras coisas legais.

O backend do blog é escrito em Perl usando o framework Mojolicious acessando uma base de dados SQLite via ORLite. Enquanto isso a camada de apresentação fica por conta do excelente Template Toolkit.

Agora que lançamos um alpha (ufa!) teremos nossas merecidas férias. A Academia chama para as famigeradas semanas de provas, pausando o nosso desenvolvimento por um breve período. Voltaremos à ativa no dia 4 de outubro, quando a partir de então, posts diários dos desenvolvedores darão início ao BBP (Big Brother Poste). Não percam!
